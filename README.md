# Portal de Conteudo Academico - Galactic Archive

## Descricao do Projeto
Este repositorio foi desenvolvido como parte de um projeto de faculdade focado no aprendizado e aplicacao pratica de tecnologias web fundamentais. O objetivo principal e demonstrar o dominio de estruturas semanticas em HTML5 e a estilizacao avancada via CSS3, com enfase em sistemas de posicionamento e arquitetura de layout.

O projeto apresenta um portal de exploracao espacial, utilizando conceitos de design de interface para criar uma experiencia de usuario organizada e visualmente distinta entre suas areas funcionais.

## Tecnologias Utilizadas
* HTML5: Estruturacao semantica de documentos.
* CSS3: Estilizacao, sistemas de cores e tipografia.

## Conceitos Implementados
O desenvolvimento atual foca nos seguintes pilares tecnicos de CSS:
* Box Model: Uso rigoroso de box-sizing: border-box para controle preciso de dimensoes.
* Posicionamento Fixo: Implementacao de cabecalho persistente durante a rolagem.
* Posicionamento Absoluto: Estruturacao de menu lateral em relacao ao fluxo do documento.
* Camadas de Profundidade: Gestao de elementos sobrepostos atraves da propriedade z-index.
* Design Responsivo Inicial: Organizacao de margens e preenchimentos para adaptabilidade de conteudo.

## Estrutura do Repositorio
* index.html: Arquivo principal com a estrutura do portal.
* style.css: Folha de estilo contendo todas as regras de layout e identidade visual.

## Roadmap de Evolucao
Este projeto esta em constante desenvolvimento e seguira o seguinte cronograma de expansao:
1. Implementacao de interatividade via JavaScript puro (DOM Manipulation).
2. Refatoracao para utilizacao de pre-processadores CSS (SASS/SCSS).
3. Migracao e modularizacao do projeto utilizando frameworks modernos como React ou Angular.
4. Integracao com APIs externas para consumo de dados em tempo real.

## Instrucoes de Execucao
Para visualizar o projeto localmente:
1. Clone o repositorio para sua maquina.
2. Navegue ate o diretorio raiz.
3. Abra o arquivo index.html em qualquer navegador web moderno.
